# JC-KZG
Java wrapper around C-KZG library: 
https://github.com/benjaminion/c-kzg/

# Repository 
The repository contains two submodules, so they need to be checked out
```shell script
> git clone <JC-KZZG repo>
> git submodule init
> git submodule update
```
